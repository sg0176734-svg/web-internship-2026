Web Dev Glossary

HTML
HyperText Markup Language — the standard structure language for web pages.
CSS
Cascading Style Sheets — controls how HTML looks visually.
JavaScript
A programming language that allows you to implement complex features on web pages.
React
A JavaScript library for building user interfaces.
API
Application Programming Interface — a way for two programs to communicate.
MERN Stack
A full-stack JavaScript framework for building web applications using MongoDB, Express, React, and Node.js.
Usage Example
Used to create a social media application where JavaScript is used for both the front-end and the back-end.
